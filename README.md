# AWS vs. Azure Cloud Cost Comparison Analysis

## 📌 Project Overview
This repository contains a comprehensive financial and architectural cost comparison between Amazon Web Services (AWS) and Microsoft Azure. The objective of this project is to analyze cloud economics, pay-as-you-go pricing dynamics, and specific operational cost drivers (compute configuration, storage architecture, and network egress) to determine the most sustainable hosting strategy for a small-scale web application.
